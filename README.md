## Description
This is a Python-based Contact Management System that allows users to store and manage contact details through a simple menu-driven interface.

The system enables users to add, view, search, update, and delete contacts efficiently.

---

## Features
- Add new contacts (Name, Phone, Email)
- View all saved contacts
- Search for a contact by name
- Update existing contact details
- Delete contacts
- Simple and user-friendly command-line interface
